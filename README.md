# ADA - Exemplo de Currículo Web (HTML & CSS)

Bem-vindas! Este repositório contém um exemplo de currículo web criado como material didático para o aprendizado de HTML e CSS. O objetivo é servir como referência e ponto de partida para que cada um crie e personalize seu próprio currículo online.

Status: Em desenvolvimento ✨

## Sobre o projeto

Este projeto foi pensado para alunos que estão iniciando em desenvolvimento web. Ele demonstra conceitos fundamentais de HTML semântico e estilização com CSS. O código é um exemplo em andamento — sinta-se livre para experimentar, modificar e melhorar.

## O que tem aqui

- `index.html` — exemplo principal do currículo em HTML.
- `semantica.html` — versão com foco em marcação semântica.
- `main.css` — estilos principais usados no projeto.
- `README.md` — este arquivo com instruções e informações do projeto.

## Projeto inspiração
[Purplefólio](https://purplefolio.framer.website/)

## Como usar

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` no navegador (duplo clique ou `File > Open`).
3. Edite o HTML e o CSS com seu editor preferido para personalizar o currículo.

Dica rápida: para ver mudanças ao salvar o arquivo, abra o `index.html` e atualize a página no navegador.

## O que aprender aqui (checklist)

- Estrutura básica de um documento HTML
- Elementos semânticos (header, nav, main, section, footer)
- Uso de classes e ids para estilização
- Box model, layout com flexbox/grid básicos
- Tipografia, cores e responsividade simples

## Boas práticas sugeridas

- Use HTML semântico sempre que possível.
- Mantenha o CSS organizado (comentários e ordem lógica).
- Priorize acessibilidade: textos alternativos em imagens, contrastes adequados e navegação por teclado.

## Como contribuir

1. Faça um fork deste repositório.
2. Crie uma branch com o seu nome: `feature/seu-nome`.
3. Faça mudanças (ex.: melhorias no layout, correções de marcação ou exercícios comentados).
4. Abra um Pull Request explicando suas alterações.

Se preferir, envie sugestões ou exemplos de currículo para que possamos incorporar no repositório principal.

## Notas para professoras

Este repositório é intencionalmente simples e aberto para personalizações. Pode ser usado em sala para demonstrar conceitos, propor exercícios (ex.: transformar o layout em responsivo) ou como base para projetos finais.
