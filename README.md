<h1>Módulo de Portfólio em HTML e CSS</h1>

Este documento apresenta o escopo e a estrutura de um projeto fictício de Portfólio Interativo, desenvolvido com foco em HTML e CSS. O projeto tem como objetivo exibir informações pessoais, habilidades e trabalhos realizados de forma responsiva, organizada e visualmente atrativa, aplicando boas práticas de design e semântica web.

- **Características Técnicas:**
  - **HTML semântico:** uso de section, article, details, summary e footer.
  - **Responsividade:** elementos com flexbox, tamanhos relativos e adaptação para telas menores.
  - **Acessibilidade:** estrutura linear, contraste adequado e uso de rótulos claros.
  - **Escalabilidade:** arquitetura modular, fácil de expandir com novas seções.

**Projeto inspiração:** [Purplefólio](https://purplefolio.framer.website/).
